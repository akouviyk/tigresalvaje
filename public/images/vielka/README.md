## Image Guidelines:

- **Quality**: High resolution, well-lit, professional appearance
- **Composition**: Clean backgrounds that don't distract from subject
- **Colors**: Rich, vibrant colors that align with website palette
- **Format**: JPG preferred for photos (smaller file size)
- **Optimization**: Compress images to keep file sizes under 500KB each

1. Place the three images in this directory (`/public/images/vielka/`)
